Welcome to Cocos

What is COCO?

Cococoin is an experimental digital meme currency that enables instant payments to anyone, anywhere in the world. Cococoin uses peer-to-peer technology to operate with no central authority: managing transactions and issuing money are carried out collectively by the network.

Cococoin Core is the name of open source software which enables the use of this currency.

Key Features

  • Secure and decentralized transactions
  • Fast and efficient peer-to-peer transfers
  • Transparent and tamper-proof ledger
  • Low transaction fees
  • Memes

Technical Features

  • 420 Coins per blocks
  • Max supply 353,152,800
  • 5 Minute block time
  • Halving every 4 years
  • Scrypt Mining Algorithm
  • More Memes
